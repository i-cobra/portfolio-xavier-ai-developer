# Xavier Ali Perez — Portfolio

A beautiful soft organic pastel portfolio built with React + Lottie animations.

## 🚀 Getting Started

### Install dependencies
```bash
npm install
```

### Run locally
```bash
npm start
```
Opens at http://localhost:3000

### Build for production
```bash
npm run build
```

## 🗂 Project Structure
```
xavier-portfolio/
├── public/
│   └── index.html
├── src/
│   ├── App.js          # Root app — renders Portfolio
│   ├── index.js        # React entry point
│   └── Portfolio.jsx   # Full portfolio component
├── package.json
└── README.md
```

## ✨ Features
- Soft Organic Pastel design with purple/blue blob backgrounds
- Lottie animations in Hero, Work, Skills, About & Contact sections
- Scroll-triggered fade-in animations via IntersectionObserver
- Fully responsive layout
- Sticky frosted-glass navigation
- Working contact form with animated success state
- Career timeline & animated skill bars

## 📦 Deploy
Works out of the box on **Vercel**, **Netlify**, or **GitHub Pages**.

Vercel (recommended):
```bash
npm install -g vercel
vercel
```
